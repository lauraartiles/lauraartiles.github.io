Mr. Spirit Pageant iOS Application

Project Description

In order to make a philanthropy event on campus more interactive, two other developers and I teamed up to develop an iOS application which would allow users to view the candidates for the pageant, vote for their favorite, donate to our philathropies, and view a Twitter feed for the event.

Tools Used

  • Sketch
  • Braintree API
  • Twitter API
  • Firebase
